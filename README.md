# Execução do projeto

## Clonar o repositório

```bash
git clone https://github.com/Carlos-Augusto7877/avaliacao-poo-sistema-atm.git
cd avaliacao-poo-sistema-atm
```

## Executar

```bash
python main.py
```

Observação: é necessário ter o Python instalado (versão 3.10 ou superior).
